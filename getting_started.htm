Guía de Instalación

Requisitos Previos

  • Git para clonar el repositorio.
  • Docker y Docker Compose.
  • Node.js y npm (si no se usa Docker para el frontend).
  • Un editor de código (opcional, recomendado: VS Code).

Paso 1: Clonar el Repositorio

Abre una terminal y clona el proyecto desde GitHub o cualquier otro servidor Git.
git clone https://github.com/Jose-YC/backTesis.git
git clone https://github.com/Jose-YC/front-tesis-II.git
Navega al directorio del proyecto:
cd front-tesis-II
cd backTesis

Paso 2: Configurar el Backend (Node.js con Docker)

Si existe un archivo .env.example, cópialo y personalízalo:
cp .env.example .env
Levantar los contenedores de Docker:
docker-compose up --build
Verificar que el backend está corriendo: Accede a la URL del backend (puede ser http://localhost:<PUERTO>).

Paso 3: Configurar el Frontend (React.js)

Instala las dependencias del proyecto:
npm install
Inicia el servidor de desarrollo:
npm start
Accede a la URL del frontend: Por defecto será http://localhost:3000.
 

Paso 4: Verificar la Conexión Entre el Frontend y Backend

Asegúrate de que el frontend está configurado para comunicarse con el backend. Revisa el archivo de configuración o las variables de entorno del frontend para confirmar la URL del backend.
 

Paso 5: Acceder al Proyecto

  • Frontend: http://localhost:3000
  • Backend (API): http://localhost:<PUERTO_BACKEND>
